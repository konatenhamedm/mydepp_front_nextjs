## 🚀 Identité Visuelle

### Pourquoi Bleu + Violet ?

✅ **Professionnel** - Le bleu inspire confiance et crédibilité  
✅ **Innovant** - Le violet évoque la créativité et la modernité  
✅ **Harmonieux** - Couleurs complémentaires sur le cercle chromatique  
✅ **Tendance** - Palette très utilisée dans les applications SaaS modernes  
✅ **Accessible** - Bon contraste pour tous les utilisateurs

### Exemples d'Utilisation

```tsx
// Dégradé primaire
className="bg-gradient-to-r from-[#0052cc] to-[#003d99]"

// Dégradé secondaire
className="bg-gradient-to-r from-[#8B5CF6] to-[#7C3AED]"

// Dégradé primaire + secondaire
className="bg-gradient-to-r from-[#0052cc] to-[#8B5CF6]"

// Bouton primaire
className="bg-[#0052cc] hover:bg-[#003d99] text-white"

// Bouton secondaire
className="bg-[#8B5CF6] hover:bg-[#7C3AED] text-white"
```
